# Invoice Management System

A basic project created by SAKEC college students during our 2nd year.

## Features

- Create new invoices.
- View existing invoices.
- Basic data management for invoices, such as amount, date, and description.
- Simple and intuitive user interface.
- PDF for bill distribution.
- Excel Sheet Generation for data analysis.

## Tech Stack

- **Backend**: Python (Flask)
- **Frontend**: HTML, CSS
- **Database**: SQLite

## Setup

You need to install `wkhtmltopdf` and put its binary directory in the project. You can download the package from the link below and the project will run:

[Download wkhtmltopdf](https://wkhtmltopdf.org/downloads.html)

## Instructions

1. First, populate the database with your username and password to create the user. You can initially use the credentials already fed in the database.
   
  ![image](https://github.com/user-attachments/assets/88d0faf2-de91-411e-915c-a3f87881bef2)

3. Run the project locally:

    ```bash
    python app.py
    ```

4. Access the application via `http://127.0.0.1:5000/` in your web browser.
